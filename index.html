Local Weather

See the Pen Local Weather App (FCC) by arapl3y (@arapley) on CodePen.

This is a local weather application that makes use of two APIs (ipapi and openweathermap) to display the current weather in the user's area according to their IP address. I had a lot of fun working on this project and it allowed me to further develop my ability to use JSON and AJAX. There are still a number of minor bugs I need to fix and additional features I'd like to implement.
